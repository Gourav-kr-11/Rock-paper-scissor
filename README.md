# Rock-Paper-Scissor

[![GitHub stars](https://img.shields.io/github/stars/Gourav-kr-11/Rock-paper-scissor?style=social)](https://github.com/Gourav-kr-11/Rock-paper-scissor/stargazers) [![GitHub forks](https://img.shields.io/github/forks/Gourav-kr-11/Rock-paper-scissor?style=social)](https://github.com/Gourav-kr-11/Rock-paper-scissor/network)

A simple browser-based Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## 🚀 Quick Start

1. Clone the repo:
   ```bash
   git clone https://github.com/Gourav-kr-11/Rock-paper-scissor.git
   ```
2. Open `index.html` in your browser.

> ✅ The repo is already configured for Vercel deployment. Visiting the root URL will load the game automatically.
